Generic Lisp the Language

Manuel Simoni

Introduction

Generic Lisp is a new dialect of Lisp, influenced by Common Lisp, Kernel, and Scheme. Notable features of Generic Lisp are lexically-scoped fexprs for metaprogramming, delimited continuations for control abstraction, and reified generics for type safety.

Data and Control Flow

PROGN

progn forms* => result
Evaluates each of its operands from left to right and returns the value of the last. If there are no operands, returns void.
